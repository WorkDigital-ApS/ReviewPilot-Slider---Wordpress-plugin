== Installation ==
1. Upload the `reviewpilot-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under 'ReviewPilot Slider' in the WordPress admin area.

== Frequently Asked Questions ==
= How do I display the ReviewPilot Slider? =
You can display the ReviewPilot Slider on your website by using the shortcode `[review-slider]`.
